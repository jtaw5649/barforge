# Barforge

A Waybar module registry and package manager for curated modules.

## Status

Coming soon. This repository is in active development.

## What this is

Barforge will provide a full-stack registry, web UI, and desktop app for
browsing, installing, and managing Waybar modules.

## Get involved

For now, track development on the `dev` branch and watch this repo for
updates. Contribution guidelines will be published as we approach v1.0.0.
